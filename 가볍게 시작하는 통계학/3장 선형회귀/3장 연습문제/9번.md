(a) 모든 행렬을 포함하는 산점도는 `pairs(데이터)` 함수를 통해 생성할 수 있다.
``` R
pairs(Auto)
```
![[9-a pairs.png]]

이는 각 객체(변수)와 객체(변수) 사이의 산점도를 모두 그린 것이다. 실제로 `plot` 함수를 이용하여 그래프를 그렸을 때와 동일한 분포를 띄고 있는 것을 확인 할 수 있다. (아래는 plot 함수를 이용하여 origin과 mpg의 산점도를 그린 것이다)
``` R
plot(origin, mpg)
```
![[9-a single plot.png]]
