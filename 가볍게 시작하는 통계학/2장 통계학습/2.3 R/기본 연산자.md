## 산술연산자

타언어의 대부분의 산술연산자를 사용할 수 있다.

```r
3 + 2      # 5
7 - 4      # 3
11 * 5     # 55
16 / 5     # 3.2
16 %/% 5   # 3: 정수나눗셈
16 %% 5	   # 1: 나머지
3^2			   # 9: 거듭제곱
```

벡터단위 연산도 지원한다.

```r
> v <- 1:4
> print(v)
[1] 1 2 3 4

> v <- v + 1
> print(v)
[1] 2 3 4 5
```

## 비교, 논리연산자

타언어의 대부분의 비교, 논리연산자를 사용할 수 있다.

```r
> 3 == 5   # FALSE
> 3 != 5   # TRUE
> 3 >= 5   # FALSE

> T || F   # TRUE
> T && F   # FALSE
```

## 벡터 연산자

- in : 포함 확인
    
    벡터에서 같은 값이 있는지 확인할 때 `%in%` 연산자를 사용한다.
    
    ```r
    > "a" %in% c("a","b","c")
    [1] TRUE
    > "d" %in% c("a","b","c")
    [1] FALSE
    ```
