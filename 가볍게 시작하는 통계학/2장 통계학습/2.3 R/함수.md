반환할 값은 반드시 괄호로 감싸 주어야 한다. 

```r
func <- function(num) {
  if(num == 1) return (1)
  return (num + func(num - 1))
}
print(func(10))   # [1] 55

# 함수를 변수(객체)처럼 사용할 수 있다.
sum <- func
print(sum(10))    # [1] 55
```

주의할 점은 내장함수에 대해서 오버라이딩에 대한 경고를 하지 않는다는 것이다. 단적인 예로 위의 sum 함수는 내장함수이지만 사용자가 만든 함수로 덮어씌어 사용하였지만 경고가 발생되지 않았다.
